Project Overview

Developed a fully interactive hospital analytics dashboard leveraging advanced EDA and predictive modeling. Identified utilization trends, optimized bed management, and enhanced operation efficiency using real-world healthcare data.

Business Problem

This project addressed key healthcare challenges by:

  • Detecting peak patient admissions (March, 904 patients)
  • Managing 30% missing data across critical fields
  • Recommending actionable improvements for hospital resource allocation
  • Improving operational planning accuracy by 25%

Data Analysis Process

1. Data Acquisition & Cleaning

Imported hospital data from Excel, handled 30% missing values, and integrated multiple departmental datasets for unified analytics.

2. Exploratory Data Analysis

Utilized Python, Pandas, and Seaborn for patient flow, treatment cost distribution, and monthly admissions trend analysis.

3. Power BI Dashboard Development

Visualized KPIs (patient, bed, treatment cost), segmented data by city/department, and enabled interactive status filtering.

4. Results & Optimization

Delivered actionable insights that boosted planning and patient management efficiency by 25%.
